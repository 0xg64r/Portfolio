Terminal Portfolio

A terminal-style portfolio showcasing my skills, technologies, and contact information in an interactive CLI interface. Built with HTML, CSS, and JavaScript, this portfolio mimics a real terminal with command-line interactions.

---

Features

- Interactive terminal interface with typing animation
- Commands available:
  - `about` → View information about me
  - `skills` → List of core skills
  - `technologies` → Tools and technologies I use
  - `contact` → How to reach me
  - `clear` → Clear the terminal screen
- Command autocompletion using `Tab`
- Command history navigation using `Arrow Up` / `Arrow Down`
- Fully responsive with ASCII and OCR banners for different screen sizes
- Terminal-style aesthetics with custom fonts and colors

---

Technologies Used

- HTML5 & CSS3 → Structure and styling
- JavaScript → Terminal engine and command processing
- Font Awesome → Icons for commands
- Google Fonts / OCR-A → Terminal and OCR-style fonts

---

How to Run Locally

Open index.html in your browser:

Option 1: Double-click the index.html file.

Option 2: Right-click → “Open with” → choose Google Chrome (or any modern browser).

The terminal portfolio should load immediately with banners and interactive commands.
